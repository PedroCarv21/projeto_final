# Sistema de Reservas para Hotelaria
Controle de reservas de quartos e serviços de hotel.

## Tecnologias

- **Back-end**: javascript + node.js + framework express
- **Front-end**: typescript + framework react

Arquitetura: 
Container no docker com 5 serviços e o front

## Execução do projeto

Digite os seguinte comandos no seu Git Bash: 

1. `git clone https://github.com/PedroCarv21/projeto_final.git`.
2. `cd projeto_final`.
3. Entrar em cada pasta (cd "pasta") e realizar o comando `npm i`.
4. Após a instalação de todas as dependências, voltar para a raiz do projeto (`cd ..`) e realizar o comando `docker-compose up --build`.
5. Entrar na pasta front-gest-hotel (`cd front-gest-hotel`) e realizar o comando npm `run dev`.
